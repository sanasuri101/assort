# 06-01: Setup & Foundation

**Goal**: Initialize the React frontend project, configure tooling (Tailwind, shadcn/ui), and establish the core layout and routing.

**Depends on**: Phase 1 (API running)

## Requirements
- **Project**: Vite + React + TypeScript.
- **Styling**: Tailwind CSS + shadcn/ui.
- **Routing**: React Router v6.
- **Layout**: Sidebar navigation, Header with Provider Selector (Mock Auth).
- **API Client**: Axios instance with base URL.

## Implementation Details

### 1. Project Initialization
- Run `npm create vite@latest frontend -- --template react-ts`.
- Install `tailwindcss`, `postcss`, `autoprefixer`.
- Install `shadcn-ui`.
- Install `lucide-react`, `react-router-dom`, `axios`, `@tanstack/react-query`.

### 2. Design System
- Configure `tailwind.config.js`.
- Add `globals.css`.
- Create basic components: `Button`, `Card`, `Input`, `Table` (via shadcn CLI).

### 3. Application Shell
- `App.tsx`: Router provider.
- `layouts/DashboardLayout.tsx`: Sidebar + Main Content Area + Header.
- `contexts/AuthContext.tsx`: Mock auth state (selected provider).

## Verification
- **Manual**: Run `npm run dev`. Verify landing page loads with layout and navigation.
